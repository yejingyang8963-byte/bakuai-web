## What "good engagement" looks like

In the comments after posting, the things that move the needle:
- Admit weaknesses honestly (Android missing, iPhone 13 Pro+ requirement)
- Reply with concrete numbers when challenged on technical claims
- Don't argue defensively — say "fair point, here's why I made that trade"
- Thank early upvoters by name

The things that get you flagged / downvoted:
- LLM-generated comments (people can tell)
- Marketing speak ("revolutionary," "game-changing")
- Defensive replies to criticism
- Hyping yourself in 3rd person

If you're going to be busy that day, **don't post** — these channels punish drop-and-run.
